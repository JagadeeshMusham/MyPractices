Selection Sort - Finds the smallest element from right array and moves it to the front of right array, uses holePosition.
    i.e.,First find the minimum value from the right side and swaps with current position.

Bubble Sort - repeatedly swaps adjacent elements to bubble up high values.

Insertion Sort - Builds a sorted array one element at a time from left to right, left will be sorted, right will be unsorted. Uses minIndex to insert the value.
    Get the current position value and place at appropriately in sorted array.

Merge Sort - Efficient divide-and-conquer approach.

Quick Sort - Uses partitioning to sort efficiently. i.e, uses pivot.
